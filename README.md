## 📊 Análise da Taxa CDI com Python

##  Contexto do Projeto

Este projeto foi desenvolvido como parte do meu processo de formação em Análise de Dados, com o objetivo de aplicar conceitos práticos de:

- Extração de dados via requisição HTTP
- Manipulação e tratamento com Pandas
- Visualização de dados com Seaborn
- Organização de código seguindo boas práticas (PEP8)

##  O que o projeto faz

O notebook realiza:

1. Extração automatizada da taxa CDI diretamente do site da B3
2. Armazenamento dos dados em arquivo CSV
3. Leitura e tratamento dos dados
4. Geração de visualização gráfica da evolução da taxa
   
##  Tecnologias utilizadas

- Python 3
- Pandas
- Seaborn
- Requests
- Google Colab

##  Aprendizados Técnicos

Durante este projeto, pratiquei:

- Automação de coleta de dados
- Estruturação de scripts
- Manipulação de DataFrames
- Geração de gráficos analíticos
- Organização de projeto para portfólio
